# Lotto - Five in Line

Interface de jeu de lotto réalisée en HTML/CSS pur, fidèle aux maquettes fournies.

## Caractéristiques

- **4 écrans différents** :
  1. Écran de démarrage avec compte à rebours (5)
  2. Écran "GO - FIVE IN LINE"
  3. Écran de jeu avec 5 boules rouges
  4. Écran de victoire "YOU WIN"

- **Design fidèle** :
  - Fond radial sunburst orange/jaune
  - Typographie en rouge (#d32f2f)
  - Effet de contour sur le texte (text-stroke)
  - Logo circulaire avec le numéro 10

- **Responsive** : Adapté pour mobile, tablette et desktop

- **Navigation** :
  - Clic ou toucher pour passer à l'écran suivant
  - Flèches gauche/droite du clavier
  - Swipe sur mobile

## Utilisation

Ouvrez simplement `index.html` dans votre navigateur.

## Structure

```
lotto-game/
├── index.html      # Structure HTML
├── styles.css      # Styles CSS avec gradient radial
├── script.js       # Navigation entre écrans
└── README.md       # Documentation
```

## Technologies

- HTML5 sémantique
- CSS3 (gradients, text-stroke, flexbox)
- JavaScript vanilla (navigation)
- Responsive design (media queries)
